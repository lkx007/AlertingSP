Summary:

Collect and interpret data so you can view performance information across multiple VNX storage systems (block and file). Quickly generate reports including how long it takes a VNX to retrieve data for end users and the amount of data that can be stored in the VNX system. Alert consolidation from EMC VNX and shown on All Alerts Console.

Main reports:

  • Unified reporting accross both Block and File world.
  • Detailed Block performance and capacity view.
  • Detailed Block capacity utilization.
  • All Alerts Console.

  • VNX Gateways VG2, VG8; OE 7.0, 7.1
  • CLARiiON CX4-120, CX4-240, CX4-480, CX4-960; FLARE R30, R31, R32
  • Celerra NX4, NS-120, NS-480, NS-960; DART 5.6, 6.0
