Summary:

Managed Service Alert configuration Solution Pack. This is a custom solution pack which installs around 200+ alert definaitons to the Alerting Backend. Most of these Alerts configuration are for Device Monitoring perspective and Generate the SNMP trap based on the alerting condition.

Main Device type Alerts included are:

  • ATMOS Alerts
  • EMC Centera Alerts
  • EMC DataDomain Alerts
  • FibreSwitch Alerts (Cisco and Brocade)
  • IBM DS Arrays Alerts
  • EMC ISILON Alerts
  • EMC RecoverPoint Alerts
  • EMC VMAX Alerts
  • EMC VNX Alerts (Both Block and File)
  • EMC VPLEX Alerts
  • VmWare vCenter Alerts
  • EMC XtremIO Alerts

Support list:

  • EMC SRM Setup >= SRM 3.6.2.0 [SRM 3.6 SP2 and above]

Requirements:

  • While installing any Solution Pack, use port 62010 as the alerting backend port - this custom service will add the DataEnricment components to Alerting Data
  • You need to add apg.manager.watchdog.timeout=900000 on SRM Primary Backend Server /opt/APG/Tools/Module-Manager//conf/module.properties file.
