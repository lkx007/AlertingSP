Summary:

Optimize service availability with The SolutionPack for EMC Isilon. Quickly see and understand performance and utilization across your EMC Isilon devices in real-time, to know which node is the bottleneck in your cluster, what are the peak/busy hours, and more.

Main reports:

  • Cluster CPU Usage
  • Cluster Throughput
  • Service Throughput
  • Disk IO
  • Connected Users
  • Capacity Planning and Trending
  • Inventory

Polled metrics:

  • Filesystems: Availability, Used, Total
  • Disks: Read/Write KB/s
  • Protocols: Latency Min/Avg/Max, KB/S read and Write, IOPS
  • CPU: Nice, Interupt, system, user
  • System: Serial, Version, Users

Support list:

  • Any EMC Isilon array
