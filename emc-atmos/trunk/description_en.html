Summary:

Access system summary reports for Resource Management Groups (RMGs), as well as, for the nodes being monitored within the RMGs. In addition, generate reports summarizing key storage parameters for the entire Atmos multi-tenant system.

Main reports:

  • System Dashboard
  • RMG Dashboard
  • Tenant Summary
  • SubTenant Summary
  • Top-N Reports

Requirements

  • Atmos Capacity Reporting Tool(ACRT) needs to be available (IP address)
  • Atmos Node running Atmos System Management Console needs to be available (IP address / Usernames for SysAdmin and TenantAdmin including passwords)

Supported models

  • Atmos version 2.1.5.0
