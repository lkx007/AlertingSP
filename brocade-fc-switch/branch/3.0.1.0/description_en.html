Summary:

Get access to performance data information that was automatically collected and interpreted (using resource grouping and mathematical calculations) from across multiple fabrics. Alert consolidation from Brocade FC Switches and shown on All Alerts Console.

Main reports:

  • Port Utilization: Report over Fibre channel ports.
  • Frames per second inbound, outbound, and discarded.
  • Throughput, inbound and outbound.
  • Port status errors.
  • Fiber Channel fabric inventory.
  • All Alerts Console.

Support list:

  • Fabric OS > V4.x

Requirements:

  • When adding a device, add it with devtype FabricSwitch
