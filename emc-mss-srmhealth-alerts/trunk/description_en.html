Summary:

Managed Service SRM Health Alert configuration Solution Pack. This is a custom solution pack which installs around 88 alert definitions. These are alert definition which triggers trap in case of SRM component failure or high resource utilization conditions.

Main SRM Health Alerts included are:

  • Alerting Backend monitoring
  • APG Portal monitoring
  • SRM Backend Service monitoring
  • SRM Collector Manager monitoring
  • SRM Database monitoring
  • SRM Event Processing Manager monitoring
  • SRM Heart Beat alert
  • SRM SNMP Collector monitoring
  • SRM VMs monitoring
  • SRM Task Scheduler monitoring
  • SRM Tomcat monitoring
  • SRM WebService Gateway monitoring
  • VMAX data collection monitoring
  • VNX data collection monitoring
  • Isilon data collection monitoring
  • Switch data collection monitoring

Support list:

  • EMC SRM Setup >= SRM 3.6.2.0 [SRM 3.6 SP2 and above]

Requirements:

  • While installing any Solution Pack, use port 62010 as the alerting backend port - this custom service will add the DataEnricment components to Alerting Data
